## Usage Guidelines

- Use primary colors for main UI elements and branding.
- Define a foreground for each new color to ensure accessibility.
- Neutral colors are versatile and can be used for text, backgrounds, and subtle UI elements.
- Use semantic colors consistently for their intended purposes (e.g., destructive for delete actions).
- Component-specific colors should be used as defined to maintain consistency across the application.

Remember to use colors consistently and consider accessibility when combining colors, especially for text using foreground colors and background combinations.
